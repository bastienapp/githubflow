# Dépôt d'entraînement GitHub Flow

1. Va sur le [dépôt d'entraînement au GitHub Flow](https://github.com/bastienapp/githubflow)
2. Va dans le projet qui a le nom de ta promo
3. Crée une *Issue* au format `nomprenom`, puis fais-en une *Pull Request*
4. Clone le projet et bascule sur ta branche
5. Crée un document au format `githubflow-nomprenom.md` contenant ton travail de recherche effectué précédemment
6. Commit et push le fichier
7. Ajoute ton binôme en *Reviewer*. Il relit, tu corriges si nécessaire, puis il valide
8. Ajoute ensuite ton formateur en *Reviewer*

Une fois toutes les étapes réalisées, tu partageras en solution :

* le lien vers la PR que tu as créée
* le lien vers la PR que tu as relue
